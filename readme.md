掌御App(IOS)
====

![](https://img.shields.io/badge/language-OC-orange.svg)


## 掌御App是什么
掌御App主要是一款用于管理测试手机的iOS App

## 掌御App功能

- 测试手机基本信息
- 测试手机使用和到期情况
- 反馈信息等

## 如何使用
- 将 PGPAppDataMacros.h 中的kNet_host 修改成自己的域名
- 添加自己的开发证书
- 修改成自己的包名
- 正式测试环境修改 
```
选中scheme：
PGPhone_dev 测试环境
PGPhone_prod 正式环境
```
